# Tripleten web_project_homeland

Este projeto tem como objetivo a criação de uma página da web responsiva. Ela conta com 4 seções: header, com o logotipo da TripleTen; lead, com título e subtítulo da página, além de uma imagem; intro, que contém uma citação, seu autor e um parágrafo de texto; e footer, com o nome da desenvolvedora da página. As linguagens usadas foram HTML e CSS e o foco principal foi adaptar o layout a diferentes tamanhos de tela.

Link para a página no GitHub Pages: https://coutomarcela.github.io/web_project_homeland/
